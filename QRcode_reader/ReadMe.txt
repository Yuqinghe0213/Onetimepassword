This application has two step, one is create a new account, the other is to generate the password.

Register process:
1. send app-debug.apk to a android mobile
2. install the application
3. open the application and input username and password
4. click "register" button to create a new account
5. use desktop application to scan the QR code
6. click "Scan your seed" and scan the QR code from desktop application

Generate one time password process:
1. input the username and password
2. click "login" button
3. click "Scan" button to scan challenge number.
4. show the one time password by QR code.
